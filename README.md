# 僵尸快跑 (Zombies Run)

## 模块一：功能与介绍

### 游戏模式
- **冒险模式**：10个关卡，渐进难度，包含多波次僵尸进攻
- **花园系统**：种植养成，1小时成熟机制，随机稀有度（普通/稀有/史诗）
- **商店系统**：购买植物皮肤、花园槽位、种子包、植物槽位
- **图鉴系统**：查看已解锁的植物和僵尸详情
- **设置系统**：背景音乐、音量控制

### 游戏特色
- 资源系统：阳光采集、金币/银币掉落、种子掉落
- 卡牌系统：7个植物槽位，可自定义选择
- 冷却系统：植物冷却、铲子功能
- 波次系统：支持多波次僵尸进攻，20秒延迟生成
- 战斗加速：支持 1x/2x 游戏速度
- 暂停功能：随时暂停/继续游戏
- 花园养成：种植、收获、出售
- 皮肤系统：植物皮肤、僵尸皮肤
- 数据持久化：localStorage 保存进度

### 音频系统
- Web Audio API 程序化音效
- 场景独立音乐：主菜单、选卡、战斗、花园
- 多种音效：射击、爆炸、收集阳光、金币掉落等

### 响应式设计
- 适配桌面端、平板、手机
- 竖屏自动提示横屏
- 横屏手机专用布局
