# Test Prompt

## 测试目标
对"僵尸快跑"游戏进行自动化 UI 测试。

## 测试场景

### 1. 主菜单界面测试
- 验证游戏标题显示正确
- 验证所有菜单按钮存在（冒险模式、花园、商店）
- 验证设置和图鉴按钮存在

### 2. 冒险模式测试
- 点击"冒险模式"按钮进入关卡选择
- 验证关卡选择界面显示
- 验证返回按钮功能

### 3. 花园系统测试
- 点击"花园"按钮
- 验证花园界面显示
- 验证返回功能

### 4. 商店系统测试
- 点击"商店"按钮
- 验证商店界面显示
- 验证金币显示
- 验证返回功能

### 5. 设置系统测试
- 点击设置按钮
- 验证音量滑块存在
- 验证返回功能

### 6. 图鉴系统测试
- 点击图鉴按钮
- 验证植物和僵尸标签页存在
- 验证返回功能

## 技术要求
- 使用 Playwright 进行端到端测试
- 启动本地 HTTP 服务器 (port 8080)
- 测试完成关闭服务器
